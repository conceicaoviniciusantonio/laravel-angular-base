# LaravelAngularBase

A generic monorepo base for building applications with Laravel + Angular. Built with a decoupled Monorepo architecture.

## 🚀 Tech Stack

- **Backend:** Laravel 12+ (PHP 8.4) - API Only
- **Frontend:** Angular 21+ (SPA) with Nebular UI
- **Database:** MySQL 8.0/9.0
- **Environment:** Docker & Docker Compose
